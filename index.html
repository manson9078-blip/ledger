Set up cloud sync ☁️

This ledger uses Firebase to sync data between you and 陳兆權 in real time — it's a free service from Google. Set it up once, and you'll both see the same records when you open the same link (or the same deployed HTML file) on your phones.

Setup steps:

1. Go to console.firebase.google.com → create a new project (free plan is fine)
2. In the left menu, "Build → Firestore Database" → create a database (test mode)
3. Next to "Project Overview" click the gear icon → Project settings → scroll to "Your apps" → add a "Web app"
4. Copy the firebaseConfig object that appears and paste it below
